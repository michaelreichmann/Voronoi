# Voronoi

*Max 8.3*

This Max Patch creates a voronoi pattern using jit.gl.pix and codebox.

The controls are:<br/>
**tempo** of the animation<br/>
**numPoints**: number of points in the pattern<br/>
**invert** the colors<br/>
